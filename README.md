## Getting Started

Follow these steps to set up the project:

1. Clone the repository:
    ```bash
    git clone https://github.com/HongLeongYong/test_copilot_agent.git
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
3. Run the application:
    ```bash
    npm start
    ```
